## 项目介绍
### TsinghuaNet
联网库，和一些其他项目需要的辅助类。支持 .NET Standard 2.0 和 .NET Standard 2.1。

### TsinghuaNet.XF
移动端版本。支持UWP、Android与iOS平台。
